# 🔑 KeyForge — Password Vault & Generator

KeyForge is a modern, ultra-secure, client-side password generator and vault. Built entirely with vanilla HTML, CSS, and JavaScript, it ensures complete privacy by keeping all data locally encrypted on your device.

## ✨ Features

- **Local Storage**: Your passwords stay in your browser's local storage (`localStorage`). No servers, no tracking, and no database leaks.
- **Master Lock Protection**: Access your vault securely with a custom master password.
- **Advanced Password Generator**: Generate highly secure passwords with custom lengths and character types (including pronounceable options).
- **Interactive Security Feedback**: Live strength analysis and entropy tracker.
- **Data Portability**: Export and import your vault securely as a JSON file.
